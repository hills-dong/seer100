刘伯温碑记是什么？

刘伯温（传） · 约1368年

刘伯温碑记（又称「救劫碑文」「太白山碑记」）据传为明朝开国军师刘伯温（刘基，1311-1375）留下的石碑预言。刘基字伯温，浙江青田人，精通天文、兵法、谶纬之学，辅佐朱元璋建立明朝，被誉为「明朝第一谋臣」，民间更将他与诸葛亮并称为中国历史上两大智者。

碑文据传刻于陕西太白山的石碑之上，在一次地震中石墙崩塌后被发现。全文以韵文体写成，主要预言末世灾劫——瘟疫、饥荒、战乱——以及劫后的太平盛世。碑文中「十愁难过猪鼠年」「若问瘟疫何时现，但看九冬十月间」等句，在2019-2020年新冠疫情期间被广泛引用和讨论。

需要特别说明的是，碑记与同样托名刘伯温的《烧饼歌》是完全不同的两部预言文本。烧饼歌以朱元璋与刘伯温的君臣问答形式，预言朝代更替和政治大事；碑记则以碑铭体裁，集中预言末世瘟疫和天灾人祸，带有强烈的宗教末世论色彩和劝善思想。碑文的真伪和成书年代在学术界存在较大争议，部分学者认为可能是后人托名之作。本站整理碑文全文并进行分段解析，供读者参考。

核心信息
「贫富若不回心转，看看死期在眼前。」——刘伯温碑记
关键段落
  • 开篇 — 天有眼地有眼，天翻地覆的末世景象
  • 瘟疫 — 若问瘟疫何时现，但看九冬十月间
  • 十愁 — 十愁难过猪鼠年，列举末世十大灾难
  • 劫年 — 铜打铁罗汉，除非善乃能保全
  • 新世 — 上元甲子到，人人哈哈笑

KFK 2075 是谁？

KFK2075 · 2018-2020

KFK 2075，即推特用户 YJ（@YJ943636），于2018年1月3日在 Twitter 注册，自称是来自2075年的时间旅行者。他以日文发布推文，回答网友关于未来的各种提问，内容涵盖科技、战争、地理变迁、社会制度等领域。

在 KFK（来自2060年）的豆瓣问答中，KFK 曾提到2075年和2062年各有一位「合法穿越者」，并称「他们的大部分回答属实，但个别回答是善意的谎言」。这使得 YJ 的预言在中文互联网上也引发了广泛关注和讨论。

YJ 在2018年至2020年间断续发布推文，声称预见了人造卫星坠落、第三次世界大战、日本首都迁移、AI 主导社会等重大事件。2018年12月一度停更并留下一串神秘乱码，2020年初短暂复出后再次消失。本站整理了其主要预言并进行了验证评估。

核心信息
「2020年将是经济和军事上非常重要的一年。现金处理的时代将在几年内结束。」
关键时间节点
  • 2018年1月 — YJ 在推特注册，首条推文自称来自2075年
  • 2018年4月 — 预言的「人造卫星坠落」事件，天宫一号坠入大气层
  • 2018年12月 — 突然停更，留下一串加密乱码
  • 2020年1月 — 短暂复出，称「2020年将是非常重要的一年」
  • 2020年5月 — 最后更新后彻底消失

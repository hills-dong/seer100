什么是梦幻时代？

Aboriginal Australian Elders · 口传传统，至少10000年历史

梦幻时代（Dreamtime / Tjukurpa / Jukurrpa）是澳洲原住民的宇宙观核心概念，指的是一个超越时间的创世纪元——过去、现在与未来在其中共存。这不是线性历史中的某个时期，而是一个永恒的、持续运作的精神维度。

在梦幻时代的叙事中，祖灵（Ancestral Beings）塑造了大地、河流、山脉和一切生灵，并通过歌之路（Songlines）将整片大陆编织成一张精神地图。这些故事不仅是创世神话，更是法律、生态知识和行为准则的载体，代代口传至今已超过一万年。

虽然原住民传统中不存在西方意义上的「预言」概念，但梦幻时代的诸多教导包含了深刻的警示：如果人类不遵守与大地的契约——不尊重火、水、动物和土地的法则——灾难将会降临。这些万年智慧在当代环境危机面前，展现出惊人的预见性。

核心信息
「我们不拥有土地，土地拥有我们。如果你不照顾大地，大地就不会照顾你。」
关键主题
  • 彩虹蛇 — 水的守护者，违背其法则将招致洪水或干旱
  • 火的法则 — 原住民用「冷火」管理土地已有数万年历史
  • 歌之路 — 穿越大陆的精神地图，记录每一处地形的故事与法则
  • 大地照护 — 人类对土地负有神圣责任，破坏土地即破坏自身
  • 陌生人的到来 — 多个部落口传中预见了外来者将带来巨变

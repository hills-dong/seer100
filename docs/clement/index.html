金·克莱门特是谁？

Kim Clement · 2006-2016

金·克莱门特（Kim Clement，1956年9月30日—2016年11月23日），出生于南非开普敦，是一位灵恩派基督教先知、音乐人和敬拜领袖。他年轻时曾深陷毒瘾，后经历信仰转变，声称获得了预言的恩赐。

克莱门特以其独特的「歌唱式预言」著称——他常在钢琴前即兴弹唱中传递预言信息。他的电视节目《预言警报》（Prophetic Alert）在全球播出，吸引了大量追随者。他的预言风格激情澎湃，常在公开聚会中当众宣告。

克莱门特最广为人知的预言是关于唐纳德·特朗普将成为美国总统——这一预言在特朗普宣布参选前数年就已发出。他还预言了科学突破、经济变化和中东局势等。2016年11月，克莱门特在特朗普当选后不久去世，享年60岁。

核心信息
「特朗普将成为总统。上帝说：我将把他放在那里，他会在两个任期内成为不寻常的总统。」
关键时间节点
  • 1956年 — 出生于南非开普敦
  • 1974年 — 信仰转变，走出毒瘾，开始事工
  • 2006年 — 首次公开预言特朗普将成为总统
  • 2007年 — 预言美国将出现两任总统
  • 2014年 — 再次确认特朗普总统预言
  • 2016年11月 — 特朗普当选后不久去世

约翰·迪是谁？

John Dee · 1564-1607

约翰·迪（John Dee，1527年7月13日—1608年或1609年），英国数学家、天文学家、占星师、神秘学家，伊丽莎白一世女王的宫廷顾问。他是当时欧洲最博学的人之一，藏书量超过欧洲大多数大学图书馆，同时也是第一个将「大英帝国」概念引入英国政治话语的人。

迪的后半生转向超自然研究，与灵媒爱德华·凯利合作进行「天使对话」——通过水晶球与天使沟通，记录天使传授的语言和预言。他由此创建了以诺魔法（Enochian magic）体系，包括一套完整的天使语言字母表。这些对话记录后被整理为《神秘日记》和《真实而忠诚的叙述》。

迪的预言大多来自天使对话记录，涉及欧洲政治格局、宗教改革、英国国运、炼金术与灵性转变等领域。虽然许多预言以隐喻和象征语言表达，但部分预言与后来的历史事件存在值得注意的对应。本站整理了他的15条核心预言，并对可验证的预言进行了真实度评估。

核心信息
「万物皆数，宇宙之奥秘铭刻于数学的语言之中。唯有通过天使的引导，凡人方能窥见造物主的设计。」
关键时间节点
  • 1564年 — 出版《单子象形文字》，提出万物统一的神秘符号体系
  • 1581年 — 开始与灵媒巴纳巴斯·索尔进行天使对话实验
  • 1582年 — 爱德华·凯利加入，天使对话进入核心阶段
  • 1583-1589年 — 与凯利游历欧洲大陆，在波兰和波西米亚继续天使对话
  • 1588年 — 西班牙无敌舰队败亡，迪此前曾为伊丽莎白提供占星建议
  • 1607年 — 最后的天使对话记录

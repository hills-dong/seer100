格庵遗录是什么？

南师古 · 约1400年

《格庵遗录》相传为朝鲜王朝学者南师古（1509—1571）所著。南师古号格庵，又号敬庵，精通易学、风水、天文、占卜与相术，晚年官任观象监天文学教授。据说他少年时于金刚山遇一神人，神人传授秘诀，南师古遂将其记录成书，这便是《格庵遗录》的由来。

全书以汉文（韩国称为「汉字」）写成，共六十篇，以隐晦的谶纬体裁，据信预言了朝鲜半岛从壬辰倭乱到未来的重大历史事件。内容涵盖王朝兴亡、外族入侵、民族分裂与统一等主题，与韩国另一部著名预言书《郑鉴录》多有呼应。

然而学术界对《格庵遗录》的真伪争议颇大。现存最广泛流传的版本由李道殷于1977年公开，被多位学者指为伪作——书中出现了「哲学」「共产」「原子」等近现代词汇，在南师古生活的16世纪并不存在。尽管如此，《格庵遗录》在韩国民间仍具有广泛影响力，被多个宗教团体引用与解读。本站整理部分篇章内容并进行验证分析，供读者参考。

QAnon 是什么？

Q (anonymous) · 2017-2021

QAnon 是一场起源于美国的政治阴谋论运动。2017年10月28日，一个自称「Q」的匿名用户在4chan论坛的/pol/板块发帖，声称自己是美国政府中拥有最高机密权限（Q级）的内部人士，掌握一个涉及全球精英阶层的秘密计划的详情。

Q在2017年至2022年间共发布了约4,950条帖子（被称为「Q drops」），从4chan转移至8chan（后更名为8kun）。这些帖子以隐晦的暗示、反问和口号为特征，声称美国前总统特朗普正在秘密打击一个由政界、商界和媒体精英组成的「深层政府」犯罪网络。

QAnon运动在2020年美国大选期间达到顶峰，并与2021年1月6日美国国会大厦事件密切相关。运动中提出了大量具体预测，包括大规模逮捕、军事法庭和「风暴」事件等，但绝大多数均未应验。本站以学术中立态度整理了其中可验证的核心预测，并逐条进行事实核查。

关键时间节点
  • 2017年10月 — Q在4chan首次发帖，声称希拉里·克林顿即将被捕
  • 2018年 — Q转移至8chan平台，追随者社群迅速壮大
  • 2019年 — FBI将QAnon列为国内恐怖主义威胁
  • 2020年 — 运动在大选期间达到顶峰，多名QAnon支持者竞选国会议员
  • 2021年1月6日 — 美国国会大厦遭冲击，多名参与者与QAnon有关
  • 2022年11月 — Q在沉寂一年多后发布最后几条帖子

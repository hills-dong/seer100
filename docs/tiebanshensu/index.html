铁板神数是什么？

邵雍（传） · 约1700年

铁板神数是中国古代最精密的命理推算体系之一。其名取「铁板钉钉」之意——推算结果铁定不移，不可更改。该体系以人的八字（出生年月日时的天干地支）为基础，通过一套复杂的数学运算公式得出数值，再查对应的条文，据称可精确推算人一生的吉凶祸福。

铁板神数传为北宋理学家、易学大家邵雍（1011—1077）所创。邵雍精研《易经》，创立了「先天象数学」，以数理推演宇宙万物变化。但现存铁板神数的版本、算法和条文体系，实际上经过了元、明、清三代众多术数家的整理、增补与改编，成书于清代康熙至乾隆年间。全书条文多达12000余条，涵盖六亲（父母兄弟妻子）、功名仕途、婚姻子嗣、寿元归期等人生各个方面。

铁板神数在民间被誉为「天下第一神数」，与「紫微斗数」「子平八字」并称中国三大命理体系。然而学术界对其有效性争议极大：支持者举出推算精确到六亲人数和属相的案例；质疑者则认为12000余条条文覆盖面极广，通过模糊匹配和话术引导容易产生「命中」的错觉。本站精选最具代表性的条文进行解读，供读者了解这一独特的数术传统。

核心信息
「先天数定，铁板不移。」——铁板神数卷首语
关键时间节点
  • 邵雍（1011—1077） — 被尊为铁板神数的创始人，实为先天象数学奠基者
  • 元明时期 — 术数家开始整理和扩充条文体系
  • 清康熙至乾隆间 — 铁板神数成书并广泛流传，版本众多
  • 清代铁笔道人 — 整理出较为通行的版本，条文编号体系定型
  • 当代 — 命理学者尝试用现代统计方法检验其准确性，争议持续

尼古拉·特斯拉是谁？

Nikola Tesla · 1893-1943

尼古拉·特斯拉（Nikola Tesla，1856年7月10日—1943年1月7日），塞尔维亚裔美国发明家、电气工程师、机械工程师和未来学家。他是交流电系统的主要推动者，拥有超过300项专利，被誉为「创造了20世纪的人」。

特斯拉不仅是一位杰出的发明家，更是一位惊人的未来预见者。在20世纪初的多次采访和文章中，他描述了智能手机、WiFi、无人机战争、可再生能源、机器人自动化等在当时看来不可思议的技术。他1926年接受《柯里尔周刊》的采访尤为著名，其中对无线通信设备的描述与现代智能手机惊人地吻合。

与传统预言家不同，特斯拉的预言源于他对电磁学、能量传输和工程学的深刻理解。他的「预言」更准确地说是基于科学原理的技术展望，这也是他的预言应验率极高的原因。本站整理了他最具代表性的16条技术预言，并对照现代科技发展进行了验证。

核心信息
「当无线技术被完美地应用时，整个地球将变成一个巨大的大脑……我们将能够在瞬间实现彼此的通信，无论距离多远。」
关键时间节点
  • 1898年 — 在麦迪逊广场花园展示遥控船，预见无人机与机器人时代
  • 1904年 — 发表文章预言无线能量传输将改变世界
  • 1926年 — 接受《柯里尔周刊》采访，描述未来口袋大小的无线通信设备
  • 1934年 — 提出「死光」定向能武器概念
  • 1935年 — 预言无线能量将为全球提供无限清洁能源

埃德加·凯西是谁？

Edgar Cayce · 1920-1944

埃德加·凯西（Edgar Cayce，1877年3月18日—1945年1月3日），美国著名灵媒与预言家，被称为「睡眠先知」。他能在自我催眠的恍惚状态下进行「解读」（readings），对医学诊断、前世轮回、未来事件等做出详细描述。一生中留下超过14000份记录在案的解读。

凯西的预言涵盖地球变动（加州沉没、纽约被毁）、亚特兰蒂斯文明的记忆、地极转移、二战走向、中国成为基督教摇篮、俄罗斯的灵性角色等。他创立的启蒙研究协会（A.R.E.）至今仍在运作，保存着所有原始解读记录。

凯西的部分预言已被现实印证，如1929年股市崩盘、二战爆发及结束时间、以色列建国等。但也有许多预言（如加州沉没、亚特兰蒂斯重现）至今未应验。本站整理了他最重要的预言条目，并对已到期的预言进行了真实度评估。

核心信息
「未来不是固定的。它取决于人类在当下所做的选择。」
关键时间节点
  • 1877年 — 出生于美国肯塔基州霍普金斯维尔
  • 1901年 — 首次进入催眠状态进行解读
  • 1923年 — 开始做「生命解读」，涉及前世与预言
  • 1931年 — 创立启蒙研究协会（A.R.E.）
  • 1944年 — 健康恶化，停止解读
  • 1945年 — 逝世于弗吉尼亚海滩

赊刀人是谁？

赊刀人 · 约1990年代至今

赊刀人是中国民间传说中一类神秘的行商。他们背着菜刀、剪刀走街串巷，以低于市价的价格赊卖刀具，分文不收。奇特的是，他们不留姓名地址，只留下一句预言作为「收款条件」——比如「等大米涨到X元一斤的时候，我来收钱」。这些预言在当时往往显得荒诞不经，没人相信会成真。

赊刀人的记录最早可追溯至1990年代的中国农村地区，此后在多个省份持续有目击报告。他们留下的预言主要涉及粮食价格、猪肉价格、房价、气候变化等与百姓生活密切相关的话题。部分预言在若干年后确实与现实吻合，令人惊叹；也有一些至今未应验或已被证伪。

赊刀人现象是中国当代最独特的民间预言传统之一。然而需要指出的是，这些故事绝大多数通过口耳相传和网络转述流传，缺乏原始文献记录，难以独立考证其真实性。本站整理了网络上广泛流传的赊刀人预言故事，进行验证分析，供读者参考。

  • 2000年代 — 多省陆续有赊刀人目击记录，预言多涉及粮价
  • 2010年代 — 赊刀人故事在社交媒体上广泛传播，引发关注

  • 2020年代 — 新冠疫情后物价波动，赊刀人预言再度引发热议

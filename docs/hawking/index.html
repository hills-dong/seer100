斯蒂芬·霍金是谁？

Stephen Hawking · 1988-2018

斯蒂芬·威廉·霍金（Stephen William Hawking，1942年1月8日—2018年3月14日），英国理论物理学家、宇宙学家，剑桥大学卢卡斯数学教授。他在黑洞物理学和宇宙学领域做出了开创性贡献，提出了著名的霍金辐射理论，证明黑洞并非完全「黑」的。

尽管21岁时被诊断出肌萎缩侧索硬化症（ALS），霍金依然成为20世纪最具影响力的物理学家之一。他的《时间简史》销量超过2500万册，使宇宙学走入大众视野。晚年的霍金频繁就人类面临的重大威胁发出警告。

霍金的预言并非神秘主义式的预言，而是基于物理学和逻辑推演的科学性警告。他反复强调人工智能的潜在威胁、气候变化的不可逆后果、以及人类必须成为多星球物种才能长期生存。本站收录了他在著作、讲座和采访中做出的15条重要预言与警告。

核心信息
「我认为人类面临的最大威胁是人工智能。一旦它全面发展，可能意味着人类的终结。」
关键时间节点
  • 1974年 — 提出霍金辐射理论，颠覆黑洞研究
  • 1988年 — 《时间简史》出版，全球销量超2500万册
  • 2010年 — 在《大设计》中警告不要主动联系外星人
  • 2014年 — 接受BBC采访，首次公开警告AI可能毁灭人类
  • 2017年 — 警告人类必须在100年内离开地球
  • 2018年3月14日 — 霍金逝世，遗作《十问》总结毕生思考

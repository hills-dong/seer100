金陵塔碑文是什么？

刘伯温（传） · 约1918年

金陵塔碑文据传为明代开国功臣刘伯温（刘基，1311-1375）所作的预言碑文。传说1918年国军拆除南京金陵塔时，在塔基下发现了这块石碑，碑上刻有韵文预言。碑文开头便以「金陵塔，金陵塔，刘基建，介石拆」点明来历——「刘基」即刘伯温，「介石」即蒋介石。

碑文以押韵的短句写成，风格类似民间谶纬歌谣，预言内容涵盖民国时期的军阀混战、抗日战争、国共内战、新中国建立等重大历史事件。碑文行文隐晦，多用拆字、谐音、隐喻等手法，与《推背图》《烧饼歌》并称为中国民间流传最广的预言文本。

然而，金陵塔碑文的真伪在学术界争议极大。目前没有可靠的考古记录证实1918年发现碑文一事，碑文最早的可查文本出现在民国报刊上，部分学者认为这很可能是民国时期假托刘伯温之名的伪作。此外，碑文的不同版本之间存在文字差异，后半部分的解读更是众说纷纭。本站整理了碑文主要流传版本的原文，并对可考证的预言进行验证分析，供读者参考。

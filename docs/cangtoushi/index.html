藏头诗是什么？

李淳风 · 约公元650年

《藏头诗》相传为唐朝贞观年间（约公元7世纪），唐太宗李世民召太史令李淳风问国运之作。唐太宗问：「朕之天下，今稍定矣。不知何人始丧我国家？」李淳风以隐晦的诗歌和谶语作答，预言了唐朝之后历代王朝的兴亡更替。因预言以隐语「藏头」而不直言，故名「藏头诗」。

全篇采用对话体裁，唐太宗发问、李淳风作答，从唐朝武则天篡权开始，依次预言五代十国、宋、元、明、清等朝代的兴衰大事。每段预言以隐喻和字谜暗指历史人物与事件，如「止戈不离身，两目长在空」暗指「武」「瞾」二字，即武则天。

李淳风同时也是《推背图》的作者之一，两部作品在内容上互有呼应。学术界普遍认为《藏头诗》的成书年代和真伪存在争议，部分学者认为现存版本可能经过后世增补。但作为中国古代预言文化的重要组成部分，它与《推背图》《烧饼歌》并列为广为流传的预言典籍，具有重要的文化研究价值。

核心信息
「天机不可泄，泄之有殃。」——李淳风答唐太宗
关键时间节点
  • 武则天篡唐 — 预言「止戈不离身」暗指武氏篡权
  • 安史之乱 — 预言「遇杨而生，逢郭而止」
  • 宋朝建立 — 预言「真龙降世」指赵匡胤
  • 蒙元入侵 — 预言「一兀之主」暗指元朝
  • 明朝兴起 — 预言「日月并行」暗合「明」字
  • 清朝入关 — 预言「八旗常在身之主」

先知以赛亚是谁？

Prophet Isaiah · 约公元前700年

以赛亚（Isaiah）是旧约圣经中最重要的先知之一，活跃于犹大王国乌西雅王至希西家王在位期间（约公元前740-700年）。以赛亚书是旧约先知书中篇幅最长的一卷，共66章，涵盖审判、救赎与末世异象。

以赛亚的预言以弥赛亚预言最为著名，包括「以马内利」的预言（7:14）、「奇妙策士、全能的神」（9:6）、「耶西的根」（11:1）以及「受苦仆人」诗篇（52-53章）。这些预言在基督教传统中被视为对耶稣基督的预指。

除弥赛亚预言外，以赛亚还预言了巴比伦的覆灭、亚述的入侵、列国的审判以及「新天新地」的末世异象。学者通常将以赛亚书分为「第一以赛亚」（1-39章）和「第二以赛亚」（40-66章）两部分。本站整理了以赛亚书中的重要预言，并附历史验证分析。

核心信息
「因有一婴孩为我们而生，有一子赐给我们，政权必担在他的肩头上。他名称为奇妙策士、全能的神、永在的父、和平的君。」——以赛亚书 9:6
关键时间节点
  • 约前740年 — 以赛亚蒙召，在圣殿中见异象（第6章）
  • 约前735年 — 以马内利预言，亚兰-以法莲危机期间（第7章）
  • 前722年 — 北国以色列被亚述灭亡，应验审判预言
  • 前701年 — 亚述围困耶路撒冷，希西家祈祷获救（第36-37章）
  • 前539年 — 巴比伦被波斯攻陷，应验以赛亚对巴比伦的审判预言

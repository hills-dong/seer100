布拉瓦茨基是谁？

Helena Blavatsky · 1877-1891

海伦娜·彼得罗芙娜·布拉瓦茨基（Helena Petrovna Blavatsky，1831年8月12日—1891年5月8日），俄国神秘学家、灵媒、作家，神智学会（Theosophical Society）联合创始人。她是19世纪最具影响力的神秘主义者之一，其思想深刻影响了新纪元运动和现代灵性传统。

布拉瓦茨基的两部巨著《揭开伊西斯的面纱》（1877）和《秘密教义》（1888）系统阐述了她的宇宙论和人类进化理论。她提出物质由不可见的力量构成、原子可以继续分割等观点，其中一些预见在后来的科学发现中得到了印证。

她的预言涵盖科学发现、文明周期、灵性进化和人类未来等领域。本站整理了她著作和书信中的15条预言性论述，并对已可验证的部分进行了真实度评估。

核心信息
「世界没有偶然——只有人类智识尚未理解的法则。物质不过是精神的凝结，而科学终将证明古代智慧所言非虚。」
关键时间节点
  • 1875年 — 在纽约创立神智学会，开启现代神秘主义运动
  • 1877年 — 出版《揭开伊西斯的面纱》，挑战正统科学与宗教
  • 1888年 — 出版《秘密教义》，提出原子可分性等超前观点
  • 1889年 — 撰写《沉默的声音》和《神智学钥匙》
  • 1891年 — 在伦敦逝世，神智学运动在全球持续扩展

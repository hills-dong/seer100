步虚大师预言是什么？

步虚大师 · 约1900年

步虚大师预言诗据称于光绪三十年（1904年）正月初七，在北京西山碧云寺通过扶乩（一种占卜方式）降示，托名隋代高僧步虚大师。实际记录者为清末居士高静涵（号清虚道人）。全诗由一首序诗和十二节预言诗组成，以七言为主，预言清末至未来百年的中国历史大事。

诗中以隐喻手法描述了清朝覆灭、民国建立、袁世凯称帝、军阀割据、日本侵华、国共内战、新中国成立等重大历史事件。前十节与20世纪中国历史高度吻合，语言虽然隐晦但指向明确，在中国民间预言文学中占有重要地位。

学术界对步虚大师预言的真伪和成书年代存在争议。「步虚」之名取自道教仪式「步虚词」，而非实有其人。部分学者认为该文本可能成于清末民初，经过后人不断增补修改，早期已验之节或为后人所加。本站整理了序诗与十二节原文并进行验证分析，供读者参考。

核心信息
「茫茫天数本难知，惟在苍生感太虚；老僧不敢多饶舌，泄露天机恐被诛。」——步虚大师预言·结尾
关键时间节点
  • 第一节 — 清朝覆灭、宣统退位
  • 第二节 — 辛亥革命、民国建立
  • 第五节 — 日本侵华、抗日战争
  • 第七节 — 日本投降、国共内战
  • 第八节 — 新中国成立

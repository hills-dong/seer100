特伦斯·麦肯纳是谁？

Terence McKenna · 1975-2000

特伦斯·麦肯纳（Terence McKenna，1946年11月16日—2000年4月3日），美国民族植物学家、神秘主义者、迷幻体验者、作家和演说家。他在加州大学伯克利分校学习生态学与资源保护，毕业后在亚马逊丛林和东南亚进行了广泛的民族植物学田野调查。

麦肯纳最著名的理论是「时间波零点」（Timewave Zero），他基于《易经》的数学结构提出了「新奇理论」——认为宇宙中新奇性（novelty）的密度在不断加速增长，最终将在2012年12月21日达到一个无限复杂的奇点。这一理论后来与玛雅历法的终结日期产生了著名的巧合。

除了2012预言外，麦肯纳还对虚拟现实、人工智能、迷幻药物的文化复兴、语言的进化、互联网作为全球大脑等话题做出了大量前瞻性论述。他的作品包括《众神的食物》《太古的复兴》《迷幻经验》等。本站收录了他最具代表性的14条预言，并对已到期的预言进行了真实度评估。

核心信息
「文化不是你的朋友。文化是一种操作系统，而其中大部分已经过时了。」
关键时间节点
  • 1971年 — 与兄弟丹尼斯在哥伦比亚亚马逊丛林的拉乔雷拉进行迷幻实验
  • 1975年 — 出版《隐形景观》，首次提出时间波零点理论
  • 1992年 — 出版《众神的食物》，提出「石化猿猴假说」
  • 1993年 — 出版《太古的复兴》，系统阐述新奇理论
  • 1998年 — 发表著名演讲「文化不是你的朋友」
  • 2012年12月21日 — 时间波零点预言的奇点日期

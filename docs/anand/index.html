阿南德是谁？

Abhigya Anand · 2019-08-22

阿比吉亚·阿南德（Abhigya Anand）是来自印度卡纳塔克邦卡拉布拉吉的少年占星师。他自幼学习吠陀占星术（Jyotish Shastra），通过分析木星、土星、罗睺、计都等星体的运行位置来做出预测。

阿南德在2020年因一段2019年8月发布的YouTube视频而爆红——视频中他准确预测了将从2019年11月起出现一种全球性病毒。这段视频在新冠疫情期间传遍亚洲社交媒体，他因此被称为「印度神童」。他运营的YouTube频道「Conscience」拥有数百万订阅者。

阿南德的预测基于行星排列（特别是木星-土星合相、日月食周期），涵盖疫情、经济危机、地缘冲突、自然灾害等领域。部分预言得到了部分验证，但许多预测措辞模糊，可套用于多种情况。本站整理了他的30条核心预言，并对已到期的预言进行了真实度评估。

核心信息
「星相不会决定你的命运，它们只是指出可能的趋势。人类的自由意志和善行才是改变命运的关键。」
关键时间节点
  • 2019年8月 — 发布预测全球疫情的YouTube视频
  • 2020年4月 — 预言视频在亚洲疯传，被称「印度神童」
  • 2020年12月 — 预测2021年2月将出现新灾难
  • 2021-2023年 — 持续发布基于吠陀占星的年度预测
  • 2024-2025年 — 预测经济动荡与地缘冲突加剧

阿瑟·C·克拉克是谁？

Arthur C. Clarke · 1945-2008

阿瑟·查尔斯·克拉克爵士（Sir Arthur Charles Clarke，1917年12月16日—2008年3月19日），英国科幻作家、科学家和未来学家。他与艾萨克·阿西莫夫、罗伯特·海因莱因并称为「科幻三巨头」。代表作包括《2001太空漫游》《与拉玛相会》《童年的终结》等。

克拉克不仅是杰出的小说家，更是一位极具远见的技术预言家。1945年，他在《无线电世界》杂志发表论文，首次提出利用地球同步轨道卫星建立全球通信网络的构想——这一预言在20年后变为现实，地球同步轨道因此被称为「克拉克轨道」。

克拉克的预言以科学素养为基础，涵盖通信技术、互联网、人工智能、太空探索等诸多领域。他的许多设想已被后来的技术发展所印证。本站收录了他最具影响力的17条技术预言，并对已到期的预言进行了真实度评估。

  • 1999年 — 接受采访，预测21世纪初互联网和远程办公的普及

以西结是谁？

Ezekiel · 约公元前590年

以西结（Ezekiel，约公元前622-570年）是旧约圣经中的四大先知之一，出身祭司家庭。公元前597年，他随犹大国第一批被掳者被流放至巴比伦，在迦巴鲁河畔的犹太流亡社区中生活和传道。

在流亡期间，以西结经历了一系列惊人的异象，并将其记录为《以西结书》共48章。他的异象以宏大、奇异和象征性著称——从四活物与轮中之轮的宝座战车（Merkabah），到枯骨复生的山谷，再到新圣殿的精确蓝图。这些异象涵盖了耶路撒冷的毁灭、列国的审判、以色列的复兴等主题。

以西结书中的许多预言已在历史中得到印证，如耶路撒冷和推罗的陷落；也有一些仍被视为末世预言，如歌革与玛各之战和新圣殿的建造。本站整理了以西结书中的核心异象与预言，并对可验证的部分进行了历史评估。

核心信息
「我必将我的灵放在你们里面，你们就要活了。我将你们安置在本地，你们就知道我耶和华如此说，也如此成就了。」——以西结书37:14
关键时间节点
  • 公元前597年 — 以西结随第一批被掳者流放至巴比伦
  • 公元前593年 — 在迦巴鲁河畔领受宝座战车异象（蒙召）
  • 公元前588-586年 — 预言耶路撒冷围城与陷落，随后应验
  • 公元前573年 — 领受新圣殿异象（以西结书40-48章）
  • 公元前332年 — 亚历山大大帝攻陷推罗岛城，应验推罗预言

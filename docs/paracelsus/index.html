帕拉塞尔苏斯是谁？

Paracelsus · 1531-1538

帕拉塞尔苏斯（Paracelsus，1493年11月11日—1541年9月24日），全名特奥弗拉斯图斯·冯·霍恩海姆（Theophrastus von Hohenheim），是瑞士-德意志医师、炼金术士、占星家和神秘学家。他被誉为「毒理学之父」，开创了以化学方法治疗疾病的先河。

帕拉塞尔苏斯一生著述丰富，涉及医学、炼金术、哲学和预言。他的《预言书》（Prognostications）写于1530年代，包含对未来数百年欧洲和人类文明的预测，涵盖宗教改革、医学进步、社会变革和自然灾害等领域。

他的许多预言具有浓厚的炼金术象征和神秘主义色彩，需要结合当时的文化语境解读。本站整理了他的主要预言，并对已到期的预言进行了真实度评估。

核心信息
「万物皆有毒，无物不含毒；唯有剂量决定它是否为毒。」
关键时间节点
  • 1493年 — 出生于瑞士施维茨州艾因西德伦
  • 1516年 — 获得费拉拉大学医学博士学位
  • 1527年 — 受聘巴塞尔市医官，公开焚烧阿维森纳和盖伦的著作
  • 1531-1538年 — 撰写《预言书》等预言性著作
  • 1541年 — 在萨尔茨堡去世，享年47岁

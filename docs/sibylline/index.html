西比林预言书是什么？

Sibyls · 约公元前500年至公元7世纪

西比尔（Sibyl）是古希腊罗马世界中一类著名的女先知，她们在神灵感召下进入迷狂状态，口述关于未来的预言。最著名的西比尔包括库迈的西比尔（Cumaean Sibyl）、厄立特里亚的西比尔（Erythraean Sibyl）等，据传曾为罗马元老院提供神谕指导。

历史上存在两组「西比林」文献：一是早已失传的「西比林之书」（Sibylline Books），曾由罗马官方保管并在国家危机时查阅；二是现存的「西比林神谕」（Sibylline Oracles），共14卷，成书于公元前2世纪至公元7世纪，属犹太-基督教伪典，借西比尔之名表达末世思想。

现存的西比林神谕以六音步诗歌写成，涵盖帝国兴亡、自然灾变、末世审判与弥赛亚降临等主题。其中大量内容被认为是「事后预言」（vaticinium ex eventu），即以预言的形式追述已发生的历史事件。本站整理了其中代表性的预言条目，并对可验证的历史预言进行了评估。

核心信息
「万物的终结即将来临，审判之日将降临大地。永生的上帝将审判所有人的灵魂与行为。」
关键时间节点
  • 约前500年 — 最早的西比尔传说出现在希腊文献中
  • 前83年 — 罗马卡庇托林山火灾，原始西比林之书被焚毁
  • 前2世纪至前1世纪 — 犹太西比林神谕（第3卷）开始编纂
  • 公元1-2世纪 — 基督教化的西比林神谕大量产生
  • 公元7世纪 — 最晚期的西比林神谕文本写就

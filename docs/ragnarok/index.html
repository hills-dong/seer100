什么是诸神黄昏？

Norse Eddas · 约800年-1220年

诸神黄昏（Ragnarok）是北欧神话中关于世界末日的预言，描述了一场毁灭性的大战、众神的陨落以及世界的毁灭与重生。这一预言体系主要保存在两部重要文献中：《诗体埃达》（Poetic Edda）中的《沃尔娃的预言》（Voluspa），以及冰岛学者斯诺里·斯图鲁松（Snorri Sturluson）于1220年前后编撰的《散文埃达》（Prose Edda）。

诸神黄昏的叙事包含一系列预兆和灾难：持续三年的芬布尔之冬、道德沦丧与亲族相残、被束缚者的挣脱、世界之蛇从海底升起、火巨人苏尔特率军进攻。最终，奥丁被巨狼芬里尔吞噬、托尔与世界之蛇同归于尽、洛基和海姆达尔互相杀死，整个世界在火焰中沉入大海。

然而，诸神黄昏并非绝对的终结。预言描述了毁灭之后的新世界：大地从海中重新升起，幸存的神祇重聚，人类始祖从世界树中走出，开启新的纪元。这种「毁灭—重生」的循环观念是北欧神话最深刻的哲学主题之一。

核心信息
「太阳变黑，大地沉入海中，繁星从天穹坠落。蒸汽与火焰猛烈交织，高高的热浪舔舐苍穹。」——《沃尔娃的预言》第57节
预言叙事脉络
  • 前兆 — 道德崩坏、亲族背叛、持续三年的严冬（芬布尔之冬）
  • 解脱 — 巨狼芬里尔挣脱锁链，洛基挣脱束缚，世界之蛇浮出海面
  • 进军 — 火巨人苏尔特率军从南方进攻，死者之船从冥界启航
  • 大战 — 诸神与巨人在维格利德原野展开最终决战
  • 陨落 — 奥丁、托尔、弗雷等众神战死，世界在火焰中毁灭
  • 重生 — 大地从海中升起，新的神祇与人类开启崭新世界

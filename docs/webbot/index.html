Web Bot 是什么？

Clif High · 1997-2024

Web Bot Project 由美国程序员 Clif High 于1997年创建，最初设计用于预测股票市场走势。该系统通过网络爬虫抓取互联网上的大量文本数据，利用语言学分析（"预测语言学"）技术，试图从人们在网上使用的词汇和语气的变化中，捕捉到对未来事件的集体无意识预感。

Web Bot 的核心理论基础是：互联网上数十亿人的语言模式会在重大事件发生前出现微妙的系统性变化，这些变化可以被算法捕捉和解读。Clif High 将此称为"预测语言学"（Predictive Linguistics），认为人类的集体潜意识会通过语言泄露未来的信息。

Web Bot 声称预测了2001年911事件、2004年印度洋海啸、2008年金融危机等重大事件，但这些说法存在很大争议。批评者指出，许多预测是事后追溯性解读，措辞模糊到几乎可以匹配任何事件，且大量末日预言（如2012年灾难）完全落空。本站收录了14条代表性预测，并进行客观验证。

核心信息
「互联网是人类集体无意识的镜子。当足够多的人以特定方式改变他们的语言模式时，未来的事件就已经投射出了它的影子。」
关键时间节点
  • 1997年 — Clif High 创建 Web Bot Project，最初用于股票市场预测
  • 2001年 — 声称在911事件前检测到"大规模破坏性事件"的语言信号
  • 2008年 — 因声称预测金融危机而获得广泛关注，开始收费发布ALTA报告
  • 2012年 — 预测的末日灾难未发生，公信力受到严重质疑
  • 2017年 — 转向加密货币预测领域，声称预测比特币价格走势

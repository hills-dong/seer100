印加预言是什么？

印加预言 · 约1200年

印加预言源自南美安第斯高原的印加文明及其继承者Q'ero族长老的口传传统。印加帝国（约1200年—1533年）是前哥伦布时期美洲最大的帝国，其宇宙观以循环时间、自然崇拜和宇宙平衡（Ayni）为核心。

印加预言体系中最核心的概念是「帕查库提」（Pachacuti），意为「世界的翻转」或「时空的逆转」，指宇宙秩序的大规模重组。印加人相信宇宙运行以五个太阳纪为周期，每个纪元终结时都伴随巨大的变革与重生。

1949年，秘鲁安第斯山区的Q'ero族长老在一次学术考察中被重新发现，他们自称是印加祭司的直系后裔，保存了帕查库提预言和塔里帕伊帕查（重逢时代）等古老传统。本站整理了印加预言体系中的核心预言与传统，并对可验证的部分进行了评估。

核心信息
「当鹰与秃鹰在天空中重逢飞翔，世界将恢复平衡。」
关键时间节点
  • 约1200年 — 印加文明兴起，宇宙观与预言传统形成
  • 1533年 — 西班牙征服者摧毁印加帝国，Q'ero族隐入高山保存传统
  • 1949年 — Q'ero长老被人类学家重新发现于安第斯高山
  • 1990年代 — Q'ero长老宣布帕查库提时代已经开始
  • 2012年 — 第五太阳纪的关键转折点

龙树谅是谁？

たつき諒 · 1999

龙树谅（たつき諒，1954年12月2日—），日本漫画家，以预知梦闻名。她自称从少女时代起便频繁经历极其逼真的预知梦，梦中场景后来多次在现实中应验。1999年，她出版了漫画作品《私が見た未来》（我所看到的未来），将多年来的预知梦以漫画形式记录下来。

这部漫画的封面上赫然写着「大灾难は2011年3月」（大灾难在2011年3月），12年后的2011年3月11日，东日本大地震和海啸如期发生，造成约两万人死亡。这一精准预言让她在日本乃至全球引发轰动。

2021年，她出版了《私が見た未来 完全版》，补充了更多预知梦的详细内容，其中关于「2025年7月大灾难」的预言引发了广泛关注和讨论。本站收录了她的15条主要预言，并对已到期的预言进行了真实度评估。

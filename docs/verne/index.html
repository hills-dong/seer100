儒勒·凡尔纳是谁？

Jules Verne · 1863-1905

儒勒·凡尔纳（Jules Verne，1828年2月8日—1905年3月24日），法国小说家、诗人、剧作家，被誉为「科幻小说之父」。他是历史上作品被翻译次数最多的法语作家，也是全世界被翻译次数第二多的作家，仅次于阿加莎·克里斯蒂。

凡尔纳的「非凡旅程」系列共54部长篇小说，构建了一个以科学探索为核心的文学宇宙。他以惊人的远见描绘了许多在当时还不存在的科技：电动潜水艇、登月飞船、直升飞机、视频通话、新闻广播、太空站、深海探索等，这些构想在他去世后的数十年间逐一成为现实。

与神秘主义预言家不同，凡尔纳的「预言」源于他对当时科学技术发展趋势的深刻理解和合理外推。他大量阅读科学期刊，与工程师和科学家保持通信，将严谨的技术细节融入引人入胜的冒险故事中。本站整理了他最具前瞻性的16条科技预言，并对照现实进行了验证。

核心信息
「凡是人能想象到的事物，必定有人能将它实现。」
关键时间节点
  • 1863年 — 《二十世纪的巴黎》描绘电力驱动的现代都市，手稿被出版商拒绝
  • 1865年 — 《从地球到月球》详细设计了登月方案，发射地点选在佛罗里达
  • 1870年 — 《海底两万里》中的鹦鹉螺号成为潜水艇的文化原型
  • 1886年 — 《征服者罗比尔》描绘了旋翼飞行器，早于莱特兄弟17年
  • 1889年 — 《2889年》构想视频通话和全球新闻网络
  • 1904年 — 《世界主宰》预见了多功能战争机器的概念

乾坤万年歌是什么？

姜子牙（传）/ 邵雍 · 约公元1000年

《乾坤万年歌》是中国古代著名的预言诗之一，传为周朝开国功臣姜子牙（姜太公）所作，故又称《太公万年歌》。全诗为七言韵文，从周朝开始，依次预言历朝历代的兴亡更替，直至天下大同。然而，学术界对此说法多有质疑，因为先秦文献中并无此诗的记载。

较为可信的观点认为，此诗的实际作者可能是宋代理学家、易学大师邵雍（1011-1077），其著有《皇极经世》《梅花诗》等，精通象数学。也有学者认为此诗系明清时期后人假托古人之名编纂，因为诗中对明清之前朝代的预言极为精准，而对之后的内容则逐渐模糊——这正是后人追述历史再伪托预言的典型特征。

无论作者是谁，《乾坤万年歌》作为中国预言文学的重要作品，与《推背图》《烧饼歌》《梅花诗》并列为古代四大预言。全诗语言精练，意象丰富，以隐喻和字谜手法暗指各朝代人物与事件，展现了中国传统文化中独特的预言智慧。本站将全诗分段整理并逐段验证分析，供读者参考。

  • 第14段 — 清朝兴亡与近代变革
